科学计数法 3.14e2; 314e-2
默认是double ，后面加f 强制转换；

java.math包下面的两个有用的类：BigInteger和 BigDecimal，这两个类可以处理任意长度的数值。BigInteger实现了任意精度的整数运算。BigDecimal实现了任意精度的浮点运算。
