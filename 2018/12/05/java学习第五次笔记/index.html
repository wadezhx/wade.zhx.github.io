算术运算符

整数运算：　
  1. 如果两个操作数有一个为Long, 则结果也为long。　
  2. 没有long时，结果为int。即使操作数全为short，byte，结果也是int。　　
    浮点运算：　　
  3. 如果两个操作数有一个为double，则结果为double。　
  4. 只有两个操作数都是float，则结果才为float。　　
    取模运算：　　
    1.其操作数可以为浮点数,一般使用整数，结果是“余数”，
    “余数”符号和左边操作数相同，
    如：7%3=1，-7%3=-1，7%-3=1。

算术运算符中++，–属于一元运算符，该类运算符只需要一个操作数。
